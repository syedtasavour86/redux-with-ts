This is a practice repository to implement Redux with TypeScript without using Redux Toolkit. 

## About the Project

This project demonstrates how to:
- Set up Redux in a TypeScript environment
- Create type-safe actions and reducers
- Connect React components to Redux state
- Implement Redux middleware with TypeScript

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm start`

## Project Structure

- `/src/components` - React components
- `/src/store` - Redux store configuration
- `/src/actions` - Redux actions
- `/src/reducers` - Redux reducers
- `/src/types` - TypeScript type definitions
